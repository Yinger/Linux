# 帮助命令
* __man__ 
  * 是 manual 的缩写
    ```
    man ls
    man -a passwd
    ```
  * man 也是一条命令，分为9章，可以使用 man 命令获得 man 的帮助
    ```
    man 7 man 
    ```
* __help__
    * shell (命令解释器) 自带的命令称为内部命令，其他的是外部命令
    * 内部命令使用 help 帮助
    ```
    help cd
    ```
    * 外部命令使用 help 帮助
    ```
    ls --help
    ```
* __info__
    * 比 help 更详细，作为 help 的补充
    ```
    info ls
    ```
